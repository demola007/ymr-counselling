# YMR 2025 - Young Ministers' Retreat

A web application for managing the Young Ministers' Retreat 2025 event.

## Features

- **Convert Management**: Register and track new converts
- **Counsellor Management**: Add and manage counsellors
- **Counsellee Management**: Register and track counsellees
- **Dashboard**: View statistics and manage uploads
- **Session Booking**: Book counselling sessions

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```sh
npm install
npm run dev
```
